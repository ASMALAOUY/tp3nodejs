# 📋 Gestion d'Utilisateurs - Application Express.js

[![Node.js Version](https://img.shields.io/badge/Node.js-24.x-green.svg)](https://nodejs.org/)
[![Express.js Version](https://img.shields.io/badge/Express.js-4.x-blue.svg)](https://expressjs.com/)
[![PostgreSQL Version](https://img.shields.io/badge/PostgreSQL-15+-blue.svg)](https://www.postgresql.org/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

Une application web complète de gestion d'utilisateurs (CRUD) développée avec **Node.js**, **Express.js**, **PostgreSQL** et **EJS**. Cette application permet d'ajouter, modifier, supprimer et afficher des utilisateurs avec une interface moderne et responsive.

## ✨ Fonctionnalités

-  **CRUD complet** : Créer, Lire, Mettre à jour, Supprimer des utilisateurs
-  **Interface responsive** : Design moderne adapté à tous les écrans
-  **Sécurisé** : Requêtes paramétrées contre les injections SQL
-  **Validation des données** : Validation côté serveur avec express-validator
-  **Pool de connexions** : Gestion optimisée des connexions PostgreSQL
-  **Architecture MVC** : Code modulaire et maintenable
-  **Mobile-friendly** : Interface adaptative

##  Technologies Utilisées

### Backend
- **[Node.js](https://nodejs.org/)** - Environnement d'exécution JavaScript
- **[Express.js](https://expressjs.com/)** - Framework web pour Node.js
- **[PostgreSQL](https://www.postgresql.org/)** - Base de données relationnelle
- **[pg](https://node-postgres.com/)** - Client PostgreSQL pour Node.js

### Frontend
- **[EJS](https://ejs.co/)** - Moteur de templates
- **[CSS3](https://developer.mozilla.org/fr/docs/Web/CSS)** - Styles modernes avec Flexbox/Grid
- **[JavaScript](https://developer.mozilla.org/fr/docs/Web/JavaScript)** - Interactions côté client

### Outils de développement
- **[nodemon](https://nodemon.io/)** - Rechargement automatique
- **[dotenv](https://github.com/motdotla/dotenv)** - Gestion des variables d'environnement
- **[express-validator](https://express-validator.github.io/)** - Validation des données
